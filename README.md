# Ranktop

Ranktop is a social media website for users to create and discuss ranked lists. Users can share their top ten across various categories, such as sports, books, fashion, and more.
## Site link
[https://ranktop.net]

## Features
- **Ranked List Posts**: Users can easily create and share ranked lists.
- **Category-Based Browsing**: Posts are grouped into categories for better discovery.
- **Engagement**: Users can like and comment on posts.
- **User Profiles**: Profile pages for users to showcase their posts.
- **Groups**: Users can create their own public or private groups to share their ranked lists to a curated audience.
- **Search**: With the help of Algolia, users can instantly search for posts or view a full page of results on their query.
- **Video tool**: Content creators can use the video tool to easily create ranking videos capable of generating millions of views

## Development Status
- **Started**: May 2023
- **Completion**: The website is now complete, however it is still being refined and new features are being added, like re-ranking.
- **Solo Project**

## Tech Stack
- **Frontend**: NextJS, React, Tailwind
- **Backend**: Prisma
- **Database**: PostgreSQL
- **Hosting**: Vercel
- **Other services**: Google Analytics, Google Cloud Storage, Algolia
